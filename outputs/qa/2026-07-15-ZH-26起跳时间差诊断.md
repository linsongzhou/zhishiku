# ZH-26：计算上行波起跳时间差诊断

## 当前问题

在面积、密度、波速和局部阻抗的多轮敏感性试验中，计算 `Wup` 与实测 `Wup` 的第一个显著起跳/谷值仍存在较大的时间差。该问题比 MQ 更优先：若时间基准或相位关系错误，后续通过土阻力、阻尼、材料参数压低 MQ 都可能是在补偿错误。

## 已排除或不足以解释的问题

| 试验 | 结果 | 对起跳时间差的含义 |
| --- | --- | --- |
| 2–4 m 局部面积降低至90%/72% | MQ仅小幅变化 | 局部浅部低阻抗不是主因 |
| PHC 环形面积 `1017.88 cm²` | MQ约25.66，接近原实心面积基线25.63 | 桩身材料面积必须按PHC保留，但不单独解释起跳时间差 |
| WS由4000扫描至4400/4600/4800 m/s | MQ非单调，4800时19.99最低 | 波速影响到时，但不能凭最低MQ确定真实波速 |
| SP 2.30 / 2.80、WS=4000 | MQ约24.67 / 25.20 | 密度主要改变阻抗幅值，非主导的时间差解释 |

## 时间关系

桩端往返传播时间为 \(2L/c\)。以传感器下有效长度 `L≈36.5 m` 估算：

| 波速 | 预期 \(2L/c\) |
| ---: | ---: |
| 4000 m/s | 18.25 ms |
| 4400 m/s | 16.59 ms |
| 4600 m/s | 15.87 ms |
| 4800 m/s | 15.21 ms |

因此，若起跳差发生在接近桩端反射区，波速或有效长度会影响它；但若差异在第一个主要 `2L/c` 之前已经存在，则桩端波速、桩端土阻力和底面积都不是第一解释，应先检查力—速度相位与数据调整。

## 优先假设与验证顺序

1. **力—速度相对相位/同步问题**：`F1/F2`、`V1/V2` 通过只说明同类传感器之间一致，不能证明力通道与速度通道完全同步。应先在 PDA-S 原始记录中显示 `F` 与 `V`，放大冲击开始至第一个峰值，检查 `F` 与 `ZV` 的零点、上升沿和峰值关系。
2. **速度通道积分/零漂调整**：A12、A34、AC 是加速度偏移调整，会影响积分得到的速度和位移；它们不是水平平移工具。应先检查位移尾部是否与实测贯入度一致，再讨论其合理性。
3. **相位偏移校正（VT）**：若原始 F 与 V 在第一峰附近存在稳定的相对水平错位，才可在 PDA-S 副本中以小步长试验 `VT`（仅平移速度相对力）。每步保存数值和截图，目标是恢复早期 F 与 `ZV` 的物理比例关系，而不是压低 MQ。手册实例明确允许在存在相位偏移时相对力曲线平移速度曲线。
4. **有效长度/波速独立校核**：只有在确定原始 F/V 同步后，才以可辨识的桩端反射到时和已知传感器下长度计算 \(c=2L/\Delta t\)。若桩端反射不清晰，单击记录不足以独立反算波速。
5. **PHC 桩端条件**：确认桩端为封闭、开口或土塞后，才调整 Bottom Area 或建立相应端部模型；它影响桩端反射形态，但不应被用来解释冲击刚开始时的 F/V 相位差。

## 原始 F/V 窗口的新增观察

PDA-S 原始窗口显示 `F1/F2 OK 1.22`、`V1/V2 OK 0.99`，但在冲击早期速度曲线先达到首峰并较快回落，力曲线峰值相对滞后且拖尾更长；下方 WU 同时出现明显的早期负向响应。该图支持“早期 F/V 关系需要优先核查”，但**尚不能直接证明**存在纯粹的电子时间偏移：传感器下方约1.6 m 的 BTA 标记对应近端特征，也可能改变早期波形。

判别原则：

- 若 VT 的恒定小平移能让开始、首峰、回落三个位置同时改善，优先考虑 F/V 相位同步问题；
- 若平移只改善一个位置、却使其他早期位置变差，优先考虑近端反射、偏心或标定/积分问题，而不是继续平移；
- 无论哪种情况，都先在 PDA-S 副本验证，再更新 CAPWAP，不把时间差交给土模型自动拟合。

## 禁止事项

- 不改 `TB` 来“对齐”：手册说明 TB 是由首个桩顶力峰推定的分析起始时间，当前窗口不可直接修改。
- 不用 A12/A34/AC 充当水平移位工具。
- 不同时联动 WS、SP、E、Area、土参数和自动 AC/AF/AT 来追求低 MQ。
- 未完成原始 F/V 相位核查前，不以 4800 m/s、钢材预设或任何 MQ 最低点作工程结论。

## 下一次操作：只做原始信号核查

在 PDA-S 中打开原始 ZH-26 副本，选择 `F/V` 显示，缩小到冲击开始至首个速度峰附近；保持当前参数和自动调整不变，先截图。随后仅在副本中每次以 `0.5` 个采样间隔尝试正、负一个 VT 小步，比较早期 F 与 ZV 的对齐方向并记录。未判断相位前，不进入 CAPWAP 土模型拟合。
