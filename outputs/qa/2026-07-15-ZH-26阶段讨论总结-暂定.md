# ZH-26 阶段讨论总结（暂定）

> **重要状态：本页是当天讨论记录，不是工程结论、检测报告结论或桩身完整性判定。**
>
> 明天获得新的原始信号、桩型资料、现场信息或复算结果后，本页任何判断均可被修改或撤回。

## 1. 当前问题

目标不是追求最低 MQ，而是解释：为什么计算 Wup 与实测 Wup 的第一处显著起跳/谷值存在较大差异，以及该差异来自桩身物理反射、F/V 相位、数据调整还是模型假设。

## 2. 已确认事实

- ZH-26 只有一击，不能在同一根桩上做重复性验证。
- 原始状态栏：`F1/F2 OK 1.22`、`V1/V2 OK 0.99`；这表明同类通道一致，**不等于**力、速度两类通道没有相对相位差。
- 桩型由用户确认是 **PHC 中空管桩**。
- 当前原始输入的外周长 `1.88496 m` 对应约600 mm外径；PHC 环形材料面积试算为 `1017.88 cm²`，推算内径约480 mm、壁厚约60 mm。尺寸和桩端开闭仍待工程资料复核。
- PDA 提示 `BTA LOW 72%@1.6 m`。BTA 是检测到的波阻抗比，不是直接的截面积百分比。

## 3. 已完成试验与观察

| 试验 | 主要结果 | 暂定含义 |
| --- | --- | --- |
| 均匀实心面积基线 | MQ约25.63 | 当前初始模型不能匹配 |
| 2–4 m 90% / 72% 面积代理 | MQ约25.35 / 24.84 | 该位置和形式的局部代理对整体 MQ 作用有限；不是精确 BTA 验证 |
| WS 扫描 | 4000/4400/4600/4800 m/s 的 MQ 约为25.63/23.98/29.18/19.99 | 模型对波速敏感，但响应非单调；不能按最低 MQ 认定真实波速或材料 |
| SP 扫描 | SP=2.30 / 2.80 时 MQ约24.67 / 25.20 | 影响小于波速；两次 EM/PM 状态尚待参数页复核 |
| PHC 环形面积 | Area=1017.88 cm² 时 MQ约25.66 | 实心面积不能作为 PHC 最终几何，但面积修正本身未解决主失配 |
| 原始 F/V 与 WU 窗口 | 速度首峰较早、回落较快，力拖尾更长；WU早期有负向响应 | 需先区分 F/V 相位与近端反射，不能只调 CAPWAP 参数 |
| VT 小步尝试 | 0.5 过大；0.05 有改善 | 存在小量级相位校正候选，尚需确认方向、最优值及是否全早期段一致改善 |

## 4. 暂定判断

1. 面积、SP 和单纯土参数不是当前“起跳时间差”的第一排查方向。
2. 4800 m/s 仅是一个拟合敏感性候选，不能用于证明钢材、真实波速或最终承载力。
3. PHC 环形截面必须保留为后续物理建模的基线；Bottom Area 是否应为外包络面积、环形面积或涉及土塞，取决于桩端开闭，当前不得盲改。
4. BTA `72%@1.6 m` 是目前唯一位于足够浅位置、可在约 `0.80 ms` 产生早期反射的物理候选；它**可能**影响第一段 Wup 起跳，但不能单独解释整段曲线的统一时间错位。
5. VT=0.05 的改善提示小量级 F/V 相位因素可能存在；它与 BTA 可以同时存在，必须分开做受控试验。

## 5. 明天的受控工作顺序

1. **VT 单变量精扫**：在 PDA-S 副本中确认 `+0.05` 或 `-0.05` 的方向；围绕有效方向试 `0.02、0.04、0.05、0.06、0.08`，仅评价冲击起跳、首峰、回落是否同时改善。
2. **BTA 精确位置试验**：另存 CAPWAP 副本，采用 PHC 环形面积基线、`VT=0`，在 PM 建立 `Depth=1.60 m` 单界面；将下段阻抗临时设为基线72%（同波速/密度时可用 Area≈732.87 cm² 作数学等效），只观察首个 Wup 起跳。
3. **禁止混用变量**：VT 试验和 BTA 试验分别完成，再做必要的组合验证；不得同时改变 WS、SP、E、Area、Bottom Area、土参数和 AC/AF/AT。
4. **补充工程信息**：确认 PHC 外径、壁厚、桩端开口/封闭、是否有桩尖或土塞条件；必要时检查传感器安装、方向、标定和锤击偏心。

## 6. 暂时禁止的结论

- 不认定桩为钢材或采用钢材参数。
- 不认定 4800 m/s 为真实波速。
- 不认定 BTA=72% 就是实际面积减少28%或已证实桩损伤。
- 不根据当前单击的低 MQ 或自动拟合结果给出承载力、完整性或施工处置意见。
